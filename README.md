# SignalForge Desktop

A modern maintained fork and continuation of the original SirenSharp project for creating custom server-sided sirens and emergency vehicle audio packages for FiveM.

SignalForge modernizes the SirenSharp ecosystem with:
- improved reliability
- modern UI/UX
- enhanced validation
- fleet tooling
- LVC & LVC:Fleet integration
- project management
- cleaner exports
- long-term maintenance and updates

# Features

## Audio & Siren Management
- Create custom siren packs
- Organize tones and tone sets
- Drag-and-drop tone ordering
- Tone playback previews
- Doppler simulation previews
- Audio waveform visualization
- Auto tone categorization

## GTA V / FiveM Integration
- Build AWC archives
- Generate REL files
- Export production-ready resources
- Support for custom siren packs
- LVC compatibility
- LVC:Fleet compatibility

## Config & Fleet Tooling
- Generate `SIRENS.lua`
- Generate `fleets.xml`
- Fleet management tools
- Vehicle assignment tools
- Validation engine for IDs, XML, Lua, and references

## Importing & Reverse Engineering
Import existing:
- AWC files
- REL files
- LVC packs
- Legacy SirenSharp projects

Reverse-map existing resources into editable projects.

## Validation & Stability
- Duplicate siren ID detection
- Invalid XML detection
- Invalid Lua detection
- Missing audio reference detection
- Export validation
- Auto-fix suggestions
- Crash recovery and backup support

# Getting Started

## Download

Download the latest version from the releases page.

## Installation

1. Download the latest `.zip`
2. Extract the archive
3. Run `SignalForge.exe`

No additional installation is required for the portable version.
